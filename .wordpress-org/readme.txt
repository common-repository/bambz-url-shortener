"BAM.bz Url Shortener" is a customizable widget that offers your visitors the service of easily shorten an url without leaving your website.

== Description ==

With this widget the visitors of your website are able to shorten an url easily without leaving your site.

You can customize the following:

*   Title of the widget
*   Stylesheet / CSS
*   Full HTML
*   Error messages are localization ready

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/bambz-url-shortener` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use Design->Widgets to drag and drop the widget to your sidebar
4. Customize title, css and html like you want.

== Frequently Asked Questions ==

= Are there any serverside connection to any server =

No. This widget only uses a secure https clientside connection.
